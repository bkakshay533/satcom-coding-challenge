# Task Management API

## Tech Stack
- Java 17
- Spring Boot
- Gradle

## How to Run
- Clone the repo
- Run:
  ./gradlew bootRun
  
## API runs at:
- http://localhost:8084
  
## Features
- In-memory storage
- Validation for required fields
- Future date validation
- Filtering by status
  
## Testing (suggested)
- Run: ./gradlew test
